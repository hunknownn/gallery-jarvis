## 자세한 설명 (4000자 이내)

Gallery Jarvis는 온디바이스 AI로 스마트폰 갤러리를 자동 정리하는 앱입니다.

주요 기능:
- AI 자동 분류: 딥러닝 모델이 사진의 시각적 특징을 분석하여 비슷한 사진끼리 그룹화
- 자동 이름 생성: 날짜, 카테고리를 조합하여 그룹 이름 자동 생성 (예: "2024.06 풍경")
- 완전한 프라이버시: 모든 분석이 기기 내에서 수행, 인터넷 연결 불필요
- 직관적 UI: Material3 디자인, 다크모드, Dynamic Color 지원
- 증분 처리: 새 사진만 추가 분석하여 빠른 업데이트

이런 분들에게 추천합니다:
- 갤러리에 수천 장의 사진이 쌓여 있는 분
- 사진 정리에 시간을 쓰기 어려운 분
- 클라우드에 사진을 올리고 싶지 않은 분
- AI 기반 자동 분류를 원하지만 프라이버시가 걱정되는 분

기술 특징:
- MobileNetV3 기반 이미지 임베딩
- 점진적 클러스터링 알고리즘
- TensorFlow Lite 온디바이스 추론
- SQLite 기반 로컬 데이터 관리

Gallery Jarvis는 인터넷 권한(INTERNET)을 요청하지 않습니다.
사진 데이터는 절대 기기 외부로 전송되지 않습니다.

## 스크린샷 요구사항

Google Play 최소 2장, 권장 8장:

1. 온보딩 화면 - 앱 소개 페이지
2. 클러스터 목록 - 분류된 그룹들이 표시된 홈 화면
3. 클러스터 상세 - 그룹 내 사진 그리드
4. 처리 진행률 - 사진 분류 중인 화면
5. 다크모드 - 다크모드 적용된 홈 화면
6. 빈 상태 - 최초 실행 시 안내 화면

> 스크린샷은 실기기에서 캡처 필요
